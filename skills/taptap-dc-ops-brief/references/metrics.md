# TapTap 店铺运营指标口径速查（简化版）

本文件用于把 MCP 返回的字段，映射为运营可理解的“指标名 + 口径解释”。正式口径以官方文档为准：
[TapTap 商店运营指标口径](https://developer.taptap.cn/docs/store/operation/metrics/)

## 使用原则

- 报告正文优先用中文短名（降低认知成本），首次出现时只解释一次。
- 不确定口径时：在报告里标注“口径待确认”，并把字段原名一并展示。
- `get_current_app_store_overview` 里的 `page_view_count` 指 **详情页浏览量（PV）**。
- 不要把 `page_view_count` 解读成 DC 面板里的“TapTap 曝光量”或推荐曝光；两者不是同一口径。
- 如果用户口头说“曝光量”，优先写成“详情页访问量（PV）”，避免误解成渠道/推荐曝光。

## 常用指标（推荐在简报里优先出现）

| 简报用名           | 官方/字段语义（简述）                 | 常见字段名（可能）              |
| ------------------ | ------------------------------------- | ------------------------------- |
| 详情页访问量（PV） | 商店详情页浏览量                      | `page_view_count`               |
| 下载请求量         | 点击下载/触发下载行为的请求次数       | `download_request_count`        |
| 下载完成量         | 实际完成下载的次数                    | `download_complete_count`       |
| 预约量             | 预约行为次数                          | `reserve_count`                 |
| 广告下载&预约量    | 通过广告带来的下载/预约相关量（如有） | `ad_download_and_reserve_count` |
| PC 下载请求量      | PC 端触发下载的请求次数               | `pc_download_request_count`     |

## 评价/社区常用指标（按需）

- 评价数、好评率、评分分布、近 7 天新增评价
- 社区内容发布量、互动（回复/点赞）趋势

说明：不同接口（overview/snapshot）可能返回不同粒度（按天/按周/聚合）。简报里请明确所用周期（例如“近 7 天”）。
